# 📊 IMDb Movie Data Analysis (EDA)

## 📌 Project Overview
This project focuses on **Exploratory Data Analysis (EDA)** of an IMDb movie dataset to identify patterns, trends, and insights related to movie ratings, genres, and popularity.  

The goal of this analysis is to transform raw movie data into **meaningful insights using Python-based data analysis techniques**, helping understand audience preferences and movie performance trends.

---

## 🎯 Project Objectives
- Perform **data cleaning and preprocessing** on movie dataset
- Analyze **distribution of movie ratings**
- Identify **most common and popular movie genres**
- Explore **relationships between movie attributes**
- Create **visualizations to communicate insights effectively**

## 🔎 Data Analysis Workflow

### 1️⃣ Data Collection
Movie data was collected from **IMDb using web scraping techniques** and stored in an Excel dataset.

### 2️⃣ Data Cleaning
The dataset was prepared for analysis by:
- Handling missing values
- Removing duplicate records
- Formatting and organizing columns
- Preparing features for analysis

### 3️⃣ Exploratory Data Analysis
EDA was performed to understand the dataset and identify trends such as:
- Distribution of movie ratings
- Genre popularity
- Patterns in movie popularity
- Relationships between different movie attributes

### 4️⃣ Data Visualization
Data visualizations were created using **Matplotlib and Seaborn** to present insights clearly and effectively.

---

## 📈 Key Insights

- Movie ratings mostly fall within the **6–8 rating range**.
- Certain genres appear **more frequently and attract higher ratings**.
- Popular movies tend to have **higher vote counts and better audience engagement**.
- Genre combinations often influence overall movie performance.
